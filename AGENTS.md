# AGENTS

## Design Context

Strategic product context lives in [`PRODUCT.md`](./PRODUCT.md) (register: product, platform: web). Visual system tokens and workbench rules live in [`design.md`](./design.md). Prefer those files over inventing new brand or UI direction.

For design work with Impeccable, start from `$impeccable` (or a scoped command like `$impeccable critique` / `$impeccable polish`) so PRODUCT.md and design.md are loaded first.
